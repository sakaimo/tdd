# FizzBuzz仕様

## 入力

- コンソールから [fizzbuzz.rb n]（nは正の整数）でプログラムを実行できる
- ただし下記の場合はプログラムは何もしないで終了する
  - nに正の整数以外が与えられた場合
  - 引数が与えられなかった場合
  - 引数が複数与えられた場合

## 出力

- 1からnまで、1ずつインクリメントした整数をコンソールに出力する
- ただし
  - 整数が3で割り切れるなら整数の代わりに"Fizz"を出力する
  - 整数が5で割り切れるなら整数の代わりに"Buzz"を出力する
  - 整数が3でも5でも割り切れるなら整数の代わりに"FizzBuzz"を表示する
- 1つ出力したら改行する


## 実行例

```
$ fizzbuzz.rb 16
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
```
